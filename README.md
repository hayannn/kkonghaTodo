# KKonghaTodo
간단한 투두리스트 프로젝트
- SpringBoot를 이용한 백엔드 구현
- SpringBoot Starter thymeleaf를 이용한 간단 화면 구현

### 📍사용 기술스택
> Backend <br>
1. SpringBoot
2. MySQL

<br>

>Frontend <br>
- Bootstrap과 스프링부트 내 thymeleaf 라이브러리 사용해 간단히 화면 구현

<br>

>OS <br>
1. Windows 11
2. MacOS Sonoma 14

<br>

### 📍라이브러리
1. lombok
2. MySQL Connector
3. spring data jpa
4. spring thymeleaf
5. spring web
6. spring boot test



<br>

### 📍현재 진행 상황
1. 투두리스트 CRUD 완성
2. 기본 디자인 적용 : todo-list.html
3. 기본 디자인 수정 : todo-list.html
4. 기본 디자인 적용 : add-todo.html, edit-todo.html ✅
